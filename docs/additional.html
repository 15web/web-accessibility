Табы

Списку табов указывается атрибут role="tabs", а элементам списка атрибут role="tab"

Скрипт инициализируется на элементе с классом "tabs", а атрибут aria-controls следует указать id панели, к которой привязана вкладка

Выбраному табу указывается aria-selected="true" и tabindex="0", остальным присваевается значение false и tabindex="-1"

Скрипт автоматически изменяет атрибуты aria-selected и tabindex в зависимости от выбранной вкладки

  • Первый
  • Второй
  • Третий
  • Четвертый

Горизонтальная навигация

Навигация выполняется с помощью тега nav, также указывается aria-label с описанием.

Желательное делать навигацию в виде списка. Из текста ссылок должно быть ясно куда они ведут

Хлебные крошки

Хлебные крошки выполняются с помощью тега nav, также указывается aria-label с описанием.

В качестве разделителя между элементами списка вставляется span с атрибутом aria-hidden="true" для того, чтобы скрин-ридеры не прочитывали этот символ.

Прогресс бар

Линия прогресса указывает, что пользователь совершил действие и приложение обрабатывает его запрос. Чтобы человек с ограниченными возможностями мог получить информацию о ходе операции используются специальные атрибуты.

role="progressbar" указывает скрин-ридеру, что это индикатор прогресса. aria-valuenow - текущее значение, aria-valuemin - минимальное, aria-valuemax - максимальное, tabindex - для того, чтобы можно было сфокусироваться на элементе

Обычный прогресс бар

35%

Большой прогресс бар

45%
                
                    <div class="progress progress_big">
                        <div class="progress-val" style="width: 45%"
                             role="progressbar" aria-valuenow="45" aria-valuemin="0"
                             aria-valuemax="100" tabindex="0">
                            <div class="progress-counter">45%</div>
                        </div>
                    </div>
Кнопка Show-Hide

Для того, чтобы пользователь мог услышать изменение, при раскрытии панели используется aria-expanded

Скрипт автоматически изменяет атрибут и фокусируется на раскрытой области

Для использования добавьте кнопке класс collapse-button, и укажите в атрибуте aria-controls id элемента для управления

tabindex используется для корректной работы скрипта
