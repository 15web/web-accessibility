Как начать пользоваться?

  1. Скачать файлы стилей и JS обработчиков accessibility.css, accessibility.min.js.
  2. Подключить стили в head. 
                    
    <head>
        
        <link rel="stylesheet" href="path/to/accessibility.css">
        
    </head>
  3. Подключить js перед закрытием тега body. 
                    
            
            <script src="path/to/accessibility.min.js"></script>
            
        </body>
    </html>
  4. Добавить на сайт верстку панели ВДС.
  5. Добавить на сайт верстку необходимых UI компонентов.
  6. Готово.

Если вы хотите использовать только какой-то определнный компонент, панель или конфигурацию цветовых схем, то на соответствующих страниах вы найдете необходимые инструкции, примеры верстки и файлы JavaScript/CSS
