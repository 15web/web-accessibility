Forms

Каждый элемент формы должен должен иметь label или в особых случаях WAI-ARIA или title.

Input с label

Когда это представляется возможным, используйте label, чтобы связать текст с элементом формы

Input с скрытым label

В некоторых случаях назначение элемента визуально ясно из контекста, в этом случае label может быть скрыт, однако нужно позаботиться о других формах представления информации для таких устройств как скрин-ридеры.

Дополнительные способы верстки элемента input

Подробную информацию вы можете найти здесь: https://www.w3.org/WAI/tutorials/forms/labels/

Select

Для группировки опций в элементе select используйте optgroup.

Aria-disabled используется для обозначения disabled элемента (см. https://www.w3.org/TR/wai-aria/states_and_properties#aria-disabled ) 

                            
                                 <select class="select" name="select" title="выберите параметры">
                                    <optgroup label="цвета">
                                        <option value="1">
                                            белый
                                        </option>
                                        <option value="2" disabled aria-disabled="true">
                                            черный
                                        </option>
                                    </optgroup>
                                     <optgroup label="размер">
                                        <option value="1">
                                            большой
                                        </option>
                                        <option value="2">
                                            маленький
                                        </option>
                                    </optgroup>
                                </select>

Чекбокс

Элемент checkbox должен иметь role="checkbox", а также динамически изменяющийся aria-checked и aria-disabled, которые обозначают состояние чекбокса.

Вы можете использовать следующую разметку, а скрипт автоматически добавит нужные атрибуты. 

                            
                                <div role="group">
                                    <input id="checkbox1" type="checkbox" role="checkbox"
                                           name="checkbox1" value="чекбокс1" disabled>
                                    <label id="checkboxLabel" for="checkbox1">чекбокс1 (disabled)</label>
                                    <input id="checkbox2" type="checkbox" role="checkbox"
                                           name="checkbox2" value="чекбокс2">
                                    <label id="checkboxLabel" for="checkbox2">чекбокс2 (default)</label>
                                    <input id="checkbox2" type="checkbox" role="checkbox"
                                           name="checkbox3" value="чекбокс2" checked>
                                    <label id="checkboxLabel" for="checkbox2">чекбокс3 (checked)</label>
                                </div>

Радио

Элементы radio должны быть обернуты в div с role="radiogroup" и иметь атрибут role="radio", а также динамически изменяющийся aria-checked и aria-disabled, которые обозначают состояние.

Вы можете использовать следующую разметку, а скрипт автоматически добавит нужные атрибуты. 

                            
                                <div role="radiogroup">
                                    <input id="radio1" type="radio" role="radio" name="first" value="Первый">
                                    <label id="label1" for="radio1">Первый</label>

                                    <input id="radio2" type="radio" role="radio" name="second" value="Второй" disabled>
                                    <label id="label2" for="radio2">Второй</label>

                                    <input id="radio3" type="radio" role="radio" name="third" value="Третий" checked>
                                    <label id="label3" for="radio3">Третий</label>
                                </div>

Ссылка обычная

Если из текста ссылки не ясно куда она ведет или ее заменяет изображение, нужно использовать aria-label, чтобы позволить скрин-ридеру озвучить подробное описание [больше]

Если ссылка недоступна используйте aria-disabled="true", а также tabindex="-1" чтобы отключить возможность сфокусироваться на ней с помощью клавиатуры Ссылка disabled

                    Ссылка disabled

Таблица с одним заголовком

Атрибут scope с значением col связывает каждую ячейку в строке с заголовками с данными в столбце

Подпись к таблице:
Last Name First Name City
Phoenix Imari Henry
Zeki Rome Min
Apirka Kelly Brynn 
                    
                        <div class="typography">
                            <table>
                                <caption>Подпись к таблице:</caption>
                                <tr>
                                    <th scope="col">Last Name</th>
                                    <th scope="col">First Name</th>
                                    <th scope="col">City</th>
                                </tr>
                                <tr>
                                    <td>Phoenix</td>
                                    <td>Imari</td>
                                    <td>Henry</td>
                                </tr>
                                <tr>
                                    <td>Zeki</td>
                                    <td>Rome</td>
                                    <td>Min</td>
                                </tr>
                                <tr>
                                    <td>Apirka</td>
                                    <td>Kelly</td>
                                    <td>Brynn</td>
                                </tr>
                            </table>
                        </div>

Таблица с двумя заголовками

Атрибут scope с значением col связывает каждую ячейку в строке с заголовками с данными в столбце

Атрибут scope с значением row связывает каждую ячейку в столбце с заголовками с данными в строке

Delivery slots:
Monday Tuesday Wednesday Thursday Friday
09:00 - 11:00 Closed Open Open Closed Closed
11:00 - 13:00 Open Open Closed Closed Closed

                                    <th scope="col">Monday</th>
                                    <th scope="col">Tuesday</th>
                                    <th scope="col">Wednesday</th>
                                    <th scope="col">Thursday</th>
                                    <th scope="col">Friday</th>
                                </tr>
                                <tr>
                                    <th scope="row">09:00 - 11:00</th>
                                    <td>Closed</td>
                                    <td>Open</td>
                                    <td>Open</td>
                                    <td>Closed</td>
                                    <td>Closed</td>
                                </tr>

Таблица с неравномерными заголовками

Атрибут scope"colgroup" связывает ячейки с заголовками первого уровня со всей группой столбцов

Заголовки второго уровня связываются только с соответсвующими колонками с помощью атрибута scope="col"

Планеты:
Mars Venus
Produced Sold Produced Sold
Teddy Bears 50,000 30,000 100,000 80,000
Board Games 10,000 5,000 12,000 9,000 
                    
                        <div class="typography">
                             <table>
                                <caption>Планеты:</caption>
                                <col>
                                <colgroup span="2"></colgroup>
                                <colgroup span="2"></colgroup>
                                <tr>
                                    <td rowspan="2"></td>
                                    <th colspan="2" scope="colgroup">Mars</th>
                                    <th colspan="2" scope="colgroup">Venus</th>
                                </tr>
                                <tr>
                                    <th scope="col">Produced</th>
                                    <th scope="col">Sold</th>
                                    <th scope="col">Produced</th>
                                    <th scope="col">Sold</th>
                                </tr>
                                <tr>
                                    <th scope="row">Teddy Bears</th>
                                    <td>50,000</td>
                                    <td>30,000</td>
                                    <td>100,000</td>
                                    <td>80,000</td>
                                </tr>
                                <tr>
                                    <th scope="row">Board Games</th>
                                    <td>10,000</td>
                                    <td>5,000</td>
                                    <td>12,000</td>
                                    <td>9,000</td>
                                </tr>
                            </table>
                        </div>
