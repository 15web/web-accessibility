Описание проекта

Проект популяризирует доступность Web для людей с ограниченными возможностями.

Вашему вниманию предлагается:

  • Панель управления настройками сайта.
  • Готовая конфигурация стилей с проработанными контрастными цветовыми схемами, hover и focus эффектами и т.п.
  • Набор основных графических компонент проработанных в соответствии с требованиями доступности WCAG 2.0.


Исходный код распространяется по лицензии MIT.


Для кроссбраузерного тестирования мы используем BrowserStack.
